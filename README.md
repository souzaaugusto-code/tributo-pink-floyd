# 🎸 Projeto - Tributo Pink Floyd

Este é um projeto simples para colocar em prática meus primeiros aprendizados em desenvolvimento web.

## 🎯 Objetivos

* **HTML:** Praticar a estruturação da página com tags essenciais (headings, paragraphs, lists e containers/divs).
* **CSS:** Fixar conceitos básicos como dark mode, text-align, padding/margin e linear-gradient.

## 👨‍💻 Tecnologias

* HTML5
* CSS3
